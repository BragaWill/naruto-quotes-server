Quotes Api
====================
Simple app to generate quotes using Node.js and Express.js

# Running locally
With Node installed, install the dependencies with:
```sh
$ npm i
```

and finally run:
```sh
nodemon index.js
```
or run: 
```sh
npx nodemon index.js
```

listening: 
```
http://localhost:3456/quotes
```
